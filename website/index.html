System Requirements

  • OS: macOS 12+ or Linux
  • RAM: 8GB recommended (2GB minimum)
  • Storage: 10GB free disk space
  • Node.js: 18.0.0 or higher
  • Python: 3.9 or higher

Dependencies That Will Be Installed

  • TTS: Python text-to-speech library
  • PyTorch: AI/ML framework
  • torchaudio: Audio processing
  • poppler-utils: PDF text extraction (optional)

Installation Instructions

# Step 1: Install Node.js dependencies
$ curl -fsSL https://voicci.com/voicci-cli/install.sh | bash
# Step 2: Install Python dependencies (automatic during first run)
$ voicci --help
# Step 3: Start using Voicci
$ voicci mybook.pdf $ voicci "The Great Gatsby"
# Generate summaries (2-5% of original length)
$ voicci summary mybook.pdf $ voicci --with-summary "1984"

Manual Installation (Alternative)

$ git clone https://github.com/voicci/voicci-cli.git $ cd voicci $ npm install $ npm link $ pip3 install TTS torch torchaudio $ brew install poppler # macOS only
